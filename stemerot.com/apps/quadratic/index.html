Δευτεροβάθμια Εξίσωση

Εφαρμογή επίλυσης δευτεροβάθμιας εξίσωσης ax² + bx + c = 0




Αυτή η εφαρμογή σας επιτρέπει να λύσετε δευτεροβάθμιες εξισώσεις της μορφής ax² + bx + c = 0 χρησιμοποιώντας την γνωστή μαθηματική φόρμουλα. Συμπληρώστε τους συντελεστές a, b και c, και αν η εξίσωση έχει πραγματικές ρίζες, θα υπολογιστούν αυτόματα.

ax² + bx + c = 0

$$ x_{1,2} = \frac{-b \pm \sqrt{b^2-4ac}}{2a} $$
